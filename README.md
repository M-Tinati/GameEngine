

# موتور بازی | GameEngine



یک پروژه **Class Library** با زبان **C#** و **.NET** که هدف آن پیاده‌سازی منطق (Logic) یک بازی نقش‌آفرینی (RPG) است؛ بدون استفاده از موتورهای بازی‌سازی مانند Unity یا Unreal Engine.



این پروژه صرفاً یک بازی نیست، بلکه یک مسیر آموزشی برای یادگیری مفاهیم برنامه‌نویسی شی‌گرا، طراحی نرم‌افزار و معماری تمیز (Clean Architecture) است. در طول توسعه، قابلیت‌های مختلف به‌صورت مرحله‌به‌مرحله به پروژه اضافه می‌شوند تا هر مبحث جدید C# در یک سناریوی واقعی پیاده‌سازی شود.



---



# 🎯 هدف پروژه



- یادگیری زبان C# از طریق یک پروژه واقعی

- تمرین اصول برنامه‌نویسی شی‌گرا (OOP)

- پیاده‌سازی منطق یک بازی به‌صورت مستقل از رابط کاربری
